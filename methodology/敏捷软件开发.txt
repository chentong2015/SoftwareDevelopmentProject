软件开发方法和工具：
1. Agile 敏捷开发：
   - 通过快速迭代项目来推进项目，不断修改计划和设计
   - 将团队划分成不同的train，朝相同目标前进moving towards the same goal/objectives

2. Scrum 一种Agile方法，Agile的一种实现
   SCRUM methodologies: ceremonies like daily stand-ups, retro, demo, backlog grooming

   Other types of agile development methodology are:
   - Extreme Programming(XP)
   - Feature-driven development(FDD)
   - Lean software development

3. Agile Team 团队的构成
   - Scrum Master
   - Quality Assurance
   - Manager
   - Product Owner
   - Developers

--------------------------------------------------------
4. Planning 一个PI通常包含多个sprint
   PI计划会议通常用来定义大的Feature，讨论之后的方向
   PI计划需要将首先的用户需求转换成技术需求
   PI计划需要确定Team Capacity以便确定工作量workload

5. Sprint
   sprint通常2到3周，包含一个阶段的每个小的任务或者fix
   sprint结束的最后一天通常会有总结会议retrospective meeting

6. Jira 用于计划和展现项目进度的平台
   - defect / task
   - feature 大的功能或更新
   - user story / acceptance criteria
   - story point / estimate the time / vote
   - backlog history
