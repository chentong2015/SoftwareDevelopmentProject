Agile 敏捷软件开发:
- 通过快速迭代项目来推进项目，不断修改计划和设计
- 将团队划分成不同的train，朝相同goal/objectives目标前进

Feature-driven development(FDD) 面向业务的应用大多基于feature来驱动开发

Extreme Programming(XP) 极限编程的特点
- User case / User Story
- Pair Programming 结对编程
- Fast iteration / Acceptance tests
- Open space

Agile Team 所有人都应具有技术背景
Scrum Master, QA, Manager, Product Owner, Dev, Tech Leader/Expert

Jira Dashboard 项目进度平台
- defect 现有的缺陷
- feature 大的功能或更新
- user story 新增加的用户案例
- acceptance criteria
- story point / estimate the time / vote
- backlog history

Confluence 项目文档和wiki平台
ServiceNow 问题请求平台
Kanban     工作进度展示平台
Miro.com   工作流程白板
PowerBI 面包展示工具
Grafana dashboard 看板展示平台
