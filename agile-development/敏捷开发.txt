Agile 敏捷软件开发:
- 通过快速迭代项目来推进项目，不断修改计划和设计
- 将团队划分成不同的train，朝相同目标前进moving towards the same goal/objectives

Extreme Programming(XP) 极限编程特点
- User case / User Story
- Pair Programming
- Fast iteration / Acceptance tests
- Open space

Agile Team 团队构成: 所有人都应具有技术背景
- Scrum Master
- Quality Assurance
- Manager
- Product Owner
- Developers
- Tech Leader/Expert
